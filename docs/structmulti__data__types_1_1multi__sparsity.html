 Taken from wikipedia. For the following watrix: 
 10 20  0  0  0  0
 0  30  0 40  0  0
 0  0  50 60 70  0
 0  0  0  0  0  80
 You would access it using CSR as:
 V   = [ 10 20 30 40 50 60 70 80 ]
 COL = [  0  1  1  3  2  3  4  5 ]   
 FIN = [  0  2  4  7  8 ]
